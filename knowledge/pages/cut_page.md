# Cut Page

## Purpose

Support fast assembly, review, and rough-cut workflows.

## Applies To

- Resolve versions: 20.x, 21.x, user installed version to verify
- Edition: Free and Studio, feature availability to verify
- OS: macOS, Windows, Linux
- Automation status: Mostly UI or hybrid; scripting depends on timeline APIs

## Typical User Intents

- Quickly assemble footage.
- Review selects.
- Create a rough cut.
- Prepare a timeline for Edit page refinement.

## UI-Capable Tasks

- Assemble and trim clips.
- Review a sequence.
- Build a simple rough cut.

## Scriptable Tasks

Potentially scriptable through timeline creation and media pool APIs once confirmed.

## Tasks Requiring Human Review

- Editorial judgment, pacing, story order, and performance selection.

## Safety Notes

- Clarify whether the user wants a new rough-cut timeline or edits to an existing timeline.
- Duplicate timelines before broad automated edits.

## Verification

- Timeline exists with intended clip order and duration.
- The original timeline remains unchanged when a duplicate was requested.
