# Fusion Page

## Purpose

Support visual effects, motion graphics, compositing, and title planning.

## Applies To

- Resolve versions: 20.x, 21.x, user installed version to verify
- Edition: Free and Studio, feature availability to verify
- OS: macOS, Windows, Linux
- Automation status: Hybrid recommended; scripting support must be verified

## Typical User Intents

- Create titles.
- Build motion graphics.
- Composite elements.
- Adjust Fusion compositions.

## UI-Capable Tasks

- Build or edit Fusion node graphs.
- Preview and refine motion or compositing.
- Adjust timing and visual design.

## Scriptable Tasks

Unknown or partial until installed Developer documentation confirms available APIs.

## Tasks Requiring Human Review

- Visual design, animation timing, keying quality, compositing realism, and typography.

## Safety Notes

- Do not replace existing Fusion comps without confirmation.
- Prefer creating a new title or duplicate comp for automated setup.

## Verification

- Composition appears at intended time.
- Text, timing, and visual result match user-approved direction.
