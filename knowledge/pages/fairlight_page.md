# Fairlight Page

## Purpose

Support audio editing, mixing, loudness planning, track routing, and delivery checks.

## Applies To

- Resolve versions: 20.x, 21.x, user installed version to verify
- Edition: Free and Studio, feature availability to verify
- OS: macOS, Windows, Linux
- Automation status: UI or hybrid recommended; exact API support must be verified

## Typical User Intents

- Normalize loudness.
- Clean dialogue.
- Adjust track levels.
- Prepare audio for delivery.
- Check true peak or broadcast target.

## UI-Capable Tasks

- Inspect and adjust tracks, buses, effects, and meters.
- Measure loudness using available tools.
- Review audio quality by listening.

## Scriptable Tasks

Unknown or partial until installed Developer documentation confirms specific audio APIs.

## Tasks Requiring Human Review

- Dialogue intelligibility, mix balance, noise reduction quality, music level, and creative sound design.

## Safety Notes

- Batch level changes are high risk.
- Duplicate timeline or preserve a backup before broad audio processing.

## Verification

- Loudness and true peak meet the target.
- Audio does not clip and remains intelligible.
- Intended tracks or buses were changed, not unrelated audio.
